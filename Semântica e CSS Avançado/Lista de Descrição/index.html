Lista de Descrição

Em HTML, uma lista de descrição, ou lista de definições, é um tipo de lista que permite exibir dados em pares de termo/descrição, ou seja, uma lista onde cada item é composto por um termo e a sua definição.


Estrutura:

A lista de descrição é estruturada utilizando as seguintes tags:
dl: Define a lista de descrição no geral.
dt: Define um termo dentro da lista de descrição.
dd: Define a descrição do termo que o acompanha.
Exemplo:
Código
HTML
Linguagem de marcação para a estrutura de conteúdo das páginas web.
CSS
Linguagem para estilizar a apresentação visual das páginas web.
