Meta tag (social sharing)

Uma meta tag de social sharing (compartilhamento social) é um código HTML que fornece informações sobre uma página web para plataformas de mídia social, como Facebook, Twitter e LinkedIn. Essas informações controlam como a página é exibida quando compartilhada, incluindo o título, a descrição, a imagem e outros dados relevantes.
Em resumo, as meta tags de social sharing são como uma "ficha técnica" para sua página web, que é usada pelas plataformas de mídia social para criar uma visualização atraente e informativa quando você compartilha um link.
Como funcionam as meta tags de social sharing?

Controlam a visualização:

As meta tags permitem que você escolha quais informações serão exibidas quando alguém compartilha um link da sua página em uma rede social.

Melhoram a experiência do usuário:

Ao controlar a visualização, você garante que o conteúdo compartilhado seja relevante e atraente, aumentando as chances de cliques e engajamento.

Ajudam na otimização:

As meta tags podem ajudar a otimizar o conteúdo compartilhado para diferentes plataformas, garantindo que a página seja exibida da melhor forma possível.

Exemplos de meta tags de social sharing:

og:title: Define o título da página, que será exibido no compartilhamento.
og:description: Fornece uma descrição da página, que pode ser exibida abaixo do título.
og:image: Especifica uma imagem que será exibida no compartilhamento.
twitter:card: Define o tipo de cartão do Twitter que será usado no compartilhamento.
twitter:title: Define o título da página para o Twitter.
twitter:description: Fornece uma descrição da página para o Twitter.
twitter:image: Especifica uma imagem para o Twitter.
Por que usar meta tags de social sharing?

Melhora a visibilidade:
Ao controlar como o conteúdo é compartilhado, você pode aumentar a visibilidade da sua página.

Aumenta o engajamento:

Um compartilhamento atraente e informativo pode levar a mais cliques, curtidas e compartilhamentos.

Otimiza a experiência do usuário:

Ao controlar as informações que são exibidas, você garante que a experiência do usuário seja mais positiva.
