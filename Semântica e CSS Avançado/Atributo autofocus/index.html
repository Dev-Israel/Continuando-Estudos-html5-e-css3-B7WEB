Atributo autofocus

O atributo autofocus em HTML serve para indicar que um elemento deve receber foco automaticamente quando a página é carregada. Isso significa que, assim que a página for carregada, o elemento com autofocus será selecionado, permitindo que o usuário comece a interagir com ele imediatamente, sem precisar clicar ou usar o teclado para o focar.
