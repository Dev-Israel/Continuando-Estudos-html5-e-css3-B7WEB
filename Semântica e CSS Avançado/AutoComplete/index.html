AutoComplete

Em HTML, o atributo autocomplete permite que os navegadores preencham automaticamente campos de formulário, como nome, e-mail e endereço, com base em informações previamente inseridas pelo usuário. Isso facilita a vida do usuário e economiza tempo.

Como funciona:

  • O atributo autocomplete é adicionado a elementos input ou form.
  • Se o atributo estiver definido como on, o navegador pode tentar preencher automaticamente os campos.
  • Se o atributo estiver definido como off, o preenchimento automático é desativado.
  • O navegador usa o histórico do usuário para preencher os campos, oferecendo sugestões ou preenchendo automaticamente as informações.

    • O usuário pode optar por aceitar ou rejeitar as sugestões do navegador.
