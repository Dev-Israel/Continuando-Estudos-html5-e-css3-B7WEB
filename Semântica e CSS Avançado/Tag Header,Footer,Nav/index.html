As tags header, footer e nav são elementos semânticos no HTML5 que fornecem estrutura e significado ao conteúdo da página. O header define o cabeçalho, a área introdutória ou navegacional de uma seção ou do documento. O footer representa o rodapé, geralmente contendo informações como autoria, direitos autorais e links. E o nav é utilizado para agrupar links de navegação, como menus e outros elementos de navegação dentro do site.
