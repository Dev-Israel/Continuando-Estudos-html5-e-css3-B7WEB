Citações em html

Em HTML, citações são marcadas para indicar texto que foi extraído de outra fonte. Existem dois elementos principais para citações:

para citações de um bloco de texto (longas) e para citações em linha (curtas). O elemento é usado para indicar a referência ou fonte da citação, e o atributo cite pode ser usado para especificar um link para a fonte.

Podemos está usando as seguintes tag para fazer citações

  • Tag p: usada para adicionar ""
  • Tag blockquote: usada citações de um bloco longo ou grande: Mateus 11:28: "Venham a mim, todos vocês que estão cansados e sobrecarregados, e eu darei descanso a vocês."
