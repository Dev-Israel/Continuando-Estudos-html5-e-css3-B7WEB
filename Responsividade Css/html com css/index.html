Podemos adicionar o CSS3 de 3 Formas Sendo:

  1. Inline

    2. Quando adicionado o css STYLE="" na mesma linha do codigo html.

  2. Interno

    3. Usando no cabeçario do codigo STYLE="" dentro da tag Head.

  3. Externo

    4. Para dicionar o css externo, temos que criar um novo arquivo o titulo é opcional mais tem que terminar com .css

    Exempleo de como é:

    link rel="stylesheet" href="estilo.css"
