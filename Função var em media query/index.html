Função Var () em @Media Query

Para que serve e como funciona?

A função var() em CSS, quando usada dentro de uma media query, permite referenciar e utilizar variáveis CSS (custom properties) para definir estilos condicionados a determinadas características de dispositivos ou telas. Isso facilita a manutenção e o reuso de estilos, tornando o código mais eficiente e legível.
Como funciona:
1. Definição da variável: Uma variável CSS é definida usando uma propriedade do tipo --minha-variavel: valor;.
2. Referência na media query: A função var() é usada para referenciar a variável no CSS da media query, por exemplo: var(--minha-variavel).
3. Estilos condicionados: A media query usa a variável para definir estilos que são aplicados quando as condições da query são atendidas, como, por exemplo, a largura da tela ou a orientação do dispositivo.
Exemplo:
Código
:root { --cor-principal: blue;
--fonte-principal: Arial;
}
@media (min-width: 768px) {
body {
background-color: var(--cor-principal);
font-family: var(--fonte-principal);
}
}
 Neste exemplo, a media query aplica a cor e a fonte principal definidas como variáveis a partir do ponto de quebra de 768px, facilitando a mudança da cor e fonte em outros locais do CSS.
Em resumo, a função var() dentro de uma media query é uma ferramenta para personalizar estilos baseados em condições de dispositivo e facilitar o uso de estilos reutilizáveis.
